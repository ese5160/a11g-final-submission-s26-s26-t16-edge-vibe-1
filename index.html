Edge Vibe

Interactive IoT-based spelling and learning system using Silicon Labs SiWG917Y, Node-RED cloud dashboard, OLED feedback, wireless connectivity, and custom PCB + enclosure design.

Project Demo

Device Description

Our device is an interactive spelling and learning system designed to help users practice word recognition and spelling through visual and audio feedback.


The system integrates sensors, OLED display, cloud connectivity, custom PCB hardware, and Node-RED dashboards to create a modern educational experience.
